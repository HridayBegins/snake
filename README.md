# Snake Game

A classic Snake game built with Python and Pygame.

## Features

- **Classic Gameplay** — Eat apples, grow longer, avoid walls and yourself
- **Score Tracking** — Real-time score display with high score persistence
- **Level Progression** — Difficulty increases as you advance
- **High Score Save** — JSON file stores best score between sessions
- **Pause Function** — Space to pause and resume
- **Fullscreen Toggle** — F11 to switch between windowed and fullscreen
- **Grid Background** — Clean visual layout for precise movement

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move snake |
| SPACE | Start / Pause |
| ESC | Quit game |
| F11 | Toggle fullscreen |

## How to Run

1. Install Python and Pygame
2. Run `python snake.py`
3. Press SPACE to start

## Requirements

- Python 3.x
- Pygame library
